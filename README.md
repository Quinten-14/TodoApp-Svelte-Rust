# todo

Install `sqlx-cli` using `cargo install sqlx-cli`, then run following:

```
git clone https://github.com/knarkzel/todo
cd todo/
```

## backend

```
cd backend/
sqlx database setup
cargo run
```

## frontend

```
cd frontend/
npm install
npm run dev
```
